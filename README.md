# Architect

**Architect** é uma plataforma open source, extensível e automatizada para criação e gerenciamento de fichas de RPG (inicialmente apenas Ordem Paranormal).

---

## Pré-requisitos

- Bun `>= 1.3.9`

## Roadmap (Inicial)

- [ ] Sistema base de ficha
- [ ] Engine de regras modular
- [ ] Sistema de plugins
- [ ] Sala de jogo em tempo real
- [ ] Sistema de combate por turnos
- [ ] Integração com mapa interativo
